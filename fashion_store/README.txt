HUONG DAN CHAY PROJECT FASHION STORE BANG XAMPP

1. Giai nen thu muc fashion_store vao:
   C:\xampp\htdocs\fashion_store

2. Mo XAMPP Control Panel:
   - Start Apache
   - Start MySQL

3. Mo trinh duyet:
   http://localhost/phpmyadmin

4. Vao tab Import, chon file database.sql trong thu muc fashion_store va bam Import.

5. Mo website:
   http://localhost/fashion_store/index.php
